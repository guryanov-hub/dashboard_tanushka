### 2.1 Вибір дати

```html
<button onclick="shiftDay(-1)">‹</button>
<input type="date" id="moon-date" />
<button onclick="shiftDay(1)">›</button>
<button onclick="goToday()">Сьогодні</button>
```

**Стан:** `let selectedDate = new Date()` (встановлюється в 12:00:00 щоб уникнути UTC-зсуву).

**Логіка:**
- При зміні дати → `renderMoon()` перераховує все локально, без запитів
- `shiftDay(delta)` — переміщення на ±1 день кнопками ‹ ›
- `goToday()` скидає `selectedDate` до сьогодні та оновлює `input.value`
- `toDateVal(d)` → `d.toISOString().split('T')[0]`

---

### 2.2 Астрономічний розрахунок (локальний, без API)

```js
function calcMoonData(date) → { day, illum, waxing, signIdx, fraction }
```

**Алгоритм:**
1. Відома точка відліку: новолуння `2000-01-06T18:14:00Z`
2. Синодичний місяць: `29.53058867` діб
3. `fraction` = дробова частина кількості повних місяців від точки відліку (0..1)
4. `day` = `Math.floor(fraction * 29.53) + 1`
5. `illum` = `round((0.5 - 0.5 * cos(fraction * 2π)) * 100)` %
6. `waxing` = `fraction < 0.5`
7. Знак зодіаку: спрощена формула еклептичної довготи Місяця
   - `L = 218.316 + 13.176396 * (jd - J2000)`
   - `M = 134.963 + 13.064993 * (jd - J2000)` → переводиться в радіани
   - `lon = L + 6.289 * sin(M)`
   - `signIdx = floor(((lon % 360) + 360) % 360 / 30)`

**Точність:** похибка ~0.5–1 лунного дня. Достатньо для практичних рекомендацій.

## Блок 3 — Теніс WTA

### HTML id: `#tennis-card`

Розташований під `.main-grid` на повну ширину сторінки.

### Джерело даних

**ESPN WTA API** — безкоштовно, без ключа, CORS відкритий.

```
GET https://site.api.espn.com/apis/site/v2/sports/tennis/wta/scoreboard?dates=YYYYMMDD
```

Виводяться матчі **всіх українських гравчинь** (фільтрація за `athlete.flag.alt === 'Ukraine'`). Жодного жорстко закодованого списку прізвищ — нові гравчині підхоплюються автоматично.

### Вкладки дат (7 днів)

Горизонтальна смуга з 7 кнопками: поточна дата + 6 днів вперед. Перша вкладка підписана "Сьогодні", решта — числом і місяцем (наприклад "15 чер").

```js
let selectedTennisDate = new Date().toISOString().split('T')[0];

function buildTennisTabs()   // будує смугу вкладок
function setTennisDate(iso)  // перемикає активну дату
```

Перемикання між вкладками не робить повторного запиту — дані всіх 7 днів завантажуються одним запуском `fetchTennis()`.

### Структура ESPN API (response)

```
response.events[]             — масив турнірів
  └── .name                   — назва турніру
  └── .groupings[]            — групування (Women's Singles тощо)
       └── .competitions[]    — окремі матчі
            ├── .id           — унікальний ID матчу
            ├── .date         — ISO timestamp UTC ("2026-06-15T09:00Z")
            ├── .timeValid    — bool: false якщо час ще не підтверджений
            └── .competitors[] — два гравці
                 └── .athlete.displayName  — повне ім'я
                 └── .athlete.flag.alt     — країна ("Ukraine", "Germany" …)
```

### Фільтрація та обхід дублів

ESPN повертає **всі матчі турніру** на кожен день запиту. Для усунення дублів використовується `Set` унікальних `comp.id`:

```js
const seenIds = new Set();
// ...
if (seenIds.has(comp.id)) continue;
seenIds.add(comp.id);
```

### Конвертація часу UTC → Київ

Україна постійно на UTC+3 (з 2022 року). Функція `utcToKyiv(t)` приймає рядок `"HH:MM"` і додає 3 години:

```js
function utcToKyiv(t) {
  const m = t.match(/^(\d{1,2}):(\d{2})$/);
  if (!m) return t;
  return `${String((+m[1] + 3) % 24).padStart(2,'0')}:${m[2]}`;
}
```

Час для матчу береться з `comp.date.slice(11, 16)` **тільки якщо** `comp.timeValid === true`. Якщо час не підтверджений ESPN — показується "час уточнюється" замість неправильного значення.

### Що відображається для кожного матчу

```
Назва турніру · Гравчиня vs Суперниця (Країна)  🕐 12:00
```

- Назва турніру — сірим (`#94a3b8`)
- Гравчиня і суперниця — основний текст
- Країна суперниці — в дужках, сірим
- Час по Києву — помаранчевим; або "час уточнюється" якщо `timeValid: false`

## Автооновлення

```js
refreshAll();  // при старті
setInterval(() => { updateTime(); fetchWeather(); }, 60000);  // кожні 60 с
```

`renderMoon()` не потрапляє в setInterval — місяць оновлюється лише при зміні дати або перезавантаженні (знак зодіаку змінюється раз на 2–3 дні).

`fetchTennis()` викликається при старті та вручну кнопкою "↻ Оновити" — не в setInterval.
